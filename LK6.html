1. Introduction to Document Processing

Document processing refers to creating, editing, and formatting documents using a keyboard and computer.

Importance in Office Administration: Ensures communication, reporting, and record-keeping are accurate, organized, and professional.

2. Types of Documents

Common documents used in offices include:

3. Keyboarding in Document Processing

  • Alphabetic keys: typing main text
  • Numeric keys: typing numbers, dates, amounts
  • Symbol keys: punctuation, currency, special characters
  • Special keys:
    • Tab ‚Äì indentation or table navigation
    • Enter/Return ‚Äì new paragraphs or commands
    • Backspace/Delete ‚Äì correcting errors
    • Shift/Caps Lock ‚Äì uppercase letters and symbols

4. Formatting Basics

  • Alignment: left, right, center, or justified
  • Spacing: single, 1.5, or double spacing
  • Indentation: first-line indents, bullets, numbering
  • Consistency: uniform font, size, and margins

5. Workflow for Typing a Document

  1. Read or outline the content.
  2. Type carefully and focus on accuracy.
  3. Format the document as needed.
  4. Proofread spelling, grammar, and layout.
  5. Save and submit or print.
