Lesson 2 ‚Äì Getting Ready to Key

Learning Objectives

After completing this lesson, learners should be able to:

  • Identify the main parts of a keyboard and their functions.
  • Demonstrate proper seating, hand, and finger positions.
  • Locate the home keys, space bar, and Enter/Return key.
  • Prepare their workstation for safe and effective keyboarding.

1. Alphabet Keys

Definition: Keys with letters A‚ÄìZ.

Purpose: Used for typing words, sentences, and text.

Example: Typing ‚ÄúOffice Administration.‚Äù

2. Number Keys

Definition: Keys for numbers 0‚Äì9, located above the alphabet keys.

Purpose: Used for typing numbers in forms, data entry, or passwords.

Example: Typing the year ‚Äú2025.‚Äù

3. Function Keys (F1‚ÄìF12)

Function keys perform shortcuts and special actions depending on the program.

  • F1 ‚Äì Opens Help menu
  • F2 ‚Äì Rename a file or folder
  • F3 ‚Äì Search
  • F4 ‚Äì Alt + F4 closes a window
  • F5 ‚Äì Refresh
  • F6 ‚Äì Move cursor to address bar or next pane
  • F7 ‚Äì Spell check
  • F8 ‚Äì Boot / selection options
  • F9 ‚Äì Refresh email/document
  • F10 ‚Äì Activate menu bar
  • F11 ‚Äì Full screen
  • F12 ‚Äì Save As / Developer tools

4. Control Keys

Definition: Ctrl, Alt, and Shift keys.

Purpose: Used with other keys for commands.

Example: Ctrl + C (Copy), Ctrl + V (Paste)

5. Special Keys

  • Enter/Return: Ends a line or executes commands
  • Backspace: Deletes characters to the left
  • Tab: Moves to next tab stop
  • Caps Lock: Toggles uppercase mode
  • Delete: Deletes characters to the right

6. Navigation Keys

Definition: Arrow keys, Home, End, Page Up, Page Down.

Purpose: Move the cursor without using the mouse.

7. Numeric Keypad

Definition: A number pad on the right side of the keyboard.

Purpose: Fast numeric entry (useful in spreadsheets).

8. Space Bar

Definition: Long key at the bottom.

Purpose: Adds spaces between words.

9. Enter / Return Key

Definition: Large key on the right side.

Purpose: Starts a new line or executes a command.

10. Home Row Keys

Definition: The main resting position of the fingers.

  • Left hand: A S D F
  • Right hand: J K L ;

11. Shift Key

Purpose: Types uppercase letters and symbols.

Example: Shift + 1 = !

12. Caps Lock

Purpose: Keeps letters uppercase until turned off.

13. Tab Key

Purpose: Moves cursor to the next field or indent.

14. Backspace Key

Purpose: Deletes the character to the left.

15. Delete Key

Purpose: Deletes characters to the right or selected text.

16. Arrow Keys

Purpose: Move the cursor in four directions.

Summary

Keyboard readiness is essential for building strong typing skills. Understanding the parts of the keyboard, proper hand placement, and key functions helps students prepare for accurate and efficient keyboarding.
