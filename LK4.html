Lesson 4 ‚Äì Numeric Keys, Symbols, and Special Keys

Learning Objectives

After completing this lesson, learners should be able to:

  • Identify and type numbers on the number row and numeric keypad.
  • Recognize and type common symbols using the shift key.
  • Demonstrate proper use of special keys (Enter, Backspace, Tab, Delete, Caps Lock, Shift).
  • Improve typing accuracy with numeric and symbol keys.
  • Apply numeric and symbol key skills in document processing tasks.

1. Number Row Keys

Definition: Numbers 0‚Äì9 located above the alphabet keys.

Finger Assignments:

  • Left pinky ‚Üí 1
  • Left ring ‚Üí 2
  • Left middle ‚Üí 3
  • Left index ‚Üí 4, 5
  • Right index ‚Üí 6, 7
  • Right middle ‚Üí 8
  • Right ring ‚Üí 9
  • Right pinky ‚Üí 0

2. Numeric Keypad (Optional)

Definition: A separate block of number keys usually located on the right side of the keyboard.

Purpose: Enables fast numeric entry for accounting, spreadsheets, and forms.

Example: Typing long number sequences such as 1234567890 quickly.

3. Symbol Keys

Definition: Characters typed using Shift + number or Shift + symbol keys.

Common Shift Key Combinations:

  • ! ‚Üí Shift + 1
  • @ ‚Üí Shift + 2
  • # ‚Üí Shift + 3
  • $ ‚Üí Shift + 4
  • % ‚Üí Shift + 5
  • ^ ‚Üí Shift + 6
  • & ‚Üí Shift + 7
  • * ‚Üí Shift + 8
  • ( ‚Üí Shift + 9
  • ) ‚Üí Shift + 0

Other Common Symbols:

- _ = + [ ] { } \ | ; : ' " , < . > / ?

4. Special Keys

  • Enter/Return: Moves to the next line or executes a command.
  • Backspace: Deletes the character to the left of the cursor.
  • Delete: Deletes the character to the right of the cursor.
  • Tab: Moves the cursor to the next tab stop or field.
  • Caps Lock: Toggles uppercase letters on/off.
  • Shift: Used for capital letters and symbols.

5. Practical Importance

  • Essential for typing invoices, forms, spreadsheets, and reports.
  • Shift and symbol key accuracy ensures proper formatting.
  • Improves speed and efficiency in document processing.
